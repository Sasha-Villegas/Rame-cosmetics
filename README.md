## Getting Started
To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Sasha-Villegas/Rame-cosmetics.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the APP
   ```js
   npm start;
   ```
